# Bno085
Interface with BNO085 9DOF Fusion IMU via I2C with Elixir and Nerves.

[Datasheet](https://www.ceva-ip.com/wp-content/uploads/BNO080_085-Datasheet.pdf)

Developed on Teyleten Robot GY-BNO085 Breakout Board.
